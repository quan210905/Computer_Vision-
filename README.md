# 🔥 Game Bắn Máy Bay - Điều Khiển Bằng Tay

## 📚 Đồ án môn: Thị giác máy tính

Một trò chơi bắn máy bay 2D được phát triển bằng Python, sử dụng **MediaPipe + OpenCV** để điều khiển bằng **cử chỉ tay qua webcam**, kết hợp với **Pygame** để xử lý game.

---

## 🎮 Tính năng nổi bật

✅ Điều khiển bằng tay (Computer Vision)  
- Sử dụng webcam để di chuyển máy bay theo ngón tay  

✅ Điều khiển bằng bàn phím  
- Hỗ trợ phím mũi tên (↑ ↓ ← →)

✅ Auto Aim (Tự động ngắm)  
- Đạn tự động khóa mục tiêu gần nhất  
- Ưu tiên bắn boss khi xuất hiện  

✅ Hệ thống Level  
- 3 màn chơi với độ khó tăng dần  

✅ Boss Battle  
- Boss xuất hiện khi đạt đủ điểm  
- Có thanh máu riêng  

✅ Hiệu ứng âm thanh  
- Bắn đạn  
- Nổ khi tiêu diệt kẻ địch  

---

## 🕹️ Cách điều khiển

### 👉 Bằng tay (Webcam)
- ☝️ Ngón trỏ: Điều khiển vị trí máy bay  

### 👉 Bằng bàn phím
| Phím | Chức năng |
|------|----------|
| ↑ ↓ ← → | Di chuyển |
| SPACE | Bắt đầu game |
| P | Tạm dừng |
| R | Chơi lại |
| ESC | Thoát |
